## Install

```sh
yarn
```

## Build

```sh
cd packages/website
yarn build
```

```sh
cd packages/website
yarn start
```

## Develop

```sh
cd packages/website
yarn dev
```

## Generate disclaimer

```sh
yarn generate-disclaimer
```

## Lint and format

```sh
yarn fix
```

```sh

yarn build
yarn start & yarn wait-on http://127.0.0.1:8080

yarn test
```
